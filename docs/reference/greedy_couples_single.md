# Greedy matching without blocking

Greedy matching without blocking

## Usage

``` r
greedy_couples_single(
  left,
  right,
  left_ids,
  right_ids,
  vars,
  distance,
  weights,
  scale,
  max_distance,
  calipers,
  strategy,
  replace = FALSE,
  ratio = 1L,
  sigma = NULL
)
```

## Value

List with pairs tibble and matching info.
